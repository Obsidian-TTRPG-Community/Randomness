# Demos

Small, self-contained feature demos:

- **Randomness Inline Demo.md** — inline `rdm:` calls, reroll/lock.
- **Randomness Image Demo.md** + **portraits.ipt** + `images/` —
  tables whose items are wiki-image embeds (`![[...]]`). For real
  generated character portraits, see the portrait features in the
  plugin itself (Settings → Randomness → Install Fantasy Portrait
  Pack).

Looking for the settlement generators (shops, taverns, temples,
whole-town templates)? They moved: **Settings → Randomness →
Install Fantasy Hub content**, or browse
[`community-generators/fantasy-hub`](../community-generators/fantasy-hub).
