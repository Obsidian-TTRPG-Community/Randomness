# Start Here

Welcome! Randomness rolls dice and picks random things from
tables — right inside your notes.

There are three ways to roll. Try each one now — every example
on this page is **live**.

## 1. Roll dice in a sentence

Type a dice roll between backticks, starting with `dice:`

The goblin hits you for `dice: 2d6 + 3` damage.

Click the 🎲 to roll again. Click the 🔒 to keep the number
forever.

## 2. Roll on a table you made

This is a plain markdown table. The `^snacks` label underneath
gives it a name:

| Snack |
| ----- |
| a suspicious sandwich |
| three olives on a sword |
| cheese of unusual size |
| one single, perfect grape |

^snacks

Now roll on it: you find `rdm:[@snacks]` in your bag.

## 3. Use the dice tray

Click the **dices icon** in the left ribbon (or press Ctrl/Cmd+P
and run "Open dice tray"). Tap some dice, hit **Roll**, and watch
them tumble.
